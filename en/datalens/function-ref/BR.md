# BR



#### Syntax {#syntax}


```
BR()
```

#### Description {#description}
Adds a line break.

**Return type**: `Markup`

#### Example {#examples}

```
'Line 1' + BR() + 'Line 2'
```


#### Data source support {#data-source-support}

`ClickHouse 21.8`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.7`, `Oracle Database 12c (12.1)`, `PostgreSQL 9.3`, `YDB`.
